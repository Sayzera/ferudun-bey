Gün 8: Fonksiyonlar

Teori

Fonksiyonlar

Fonksiyonlar, belirli bir görevi yerine getiren kod bloklarıdır:

  • Parametreler: Fonksiyona veri aktarmak için kullanılır
  • Argümanlar: Fonksiyona gönderilen değerler
  • Dönüş değeri: Fonksiyonun sonucu

Fonksiyon Tanımlama:

function topla(a, b) {
    return a + b;
}
