Gün 6: Döngüler

Teori

For Döngüsü

Belirli bir koşul sağlandığı sürece tekrarlanan işlemler için kullanılır.

for (let i = 0; i < 10; i++) {
    // işlemler
}

While Döngüsü

Koşul doğru olduğu sürece çalışan döngü.

while (koşul) {
    // işlemler
}

Do-While Döngüsü

En az bir kez çalışan ve koşul doğru olduğu sürece devam eden döngü.

do {
    // işlemler
} while (koşul);

ForEach Döngüsü

Diziler üzerinde döngü yapmak için kullanılır.

dizi.forEach((eleman, index) => {
    // işlemler
});

For-In Döngüsü

Objelerin özellikleri üzerinde döngü yapmak için kullanılır.

for (let ozellik in obje) {
    // işlemler
}

For-Of Döngüsü

Yinelenebilir nesneler (diziler, stringler vb.) üzerinde döngü yapmak için kullanılır.

for (let eleman of dizi) {
    // işlemler
}
