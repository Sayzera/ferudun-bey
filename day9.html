Gün 9: Geri Dönüş Değerleri ve Fonksiyon Kapsamı

Teori

Geri Dönüş Değerleri

Fonksiyonlar return ifadesi ile değer döndürebilir:

function topla(a, b) {
    return a + b;
}

Fonksiyon Kapsamı (Scope)

Değişkenlerin erişilebilir olduğu alan:

  • Global Scope: Tüm kodda erişilebilir
  • Local Scope: Sadece fonksiyon içinde erişilebilir
  • Block Scope: Sadece blok içinde erişilebilir (let, const)
